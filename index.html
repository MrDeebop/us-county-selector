County Grouping Tool with State Data

Instructions:

  1. Upload your complete shapefile folder (must include .shp, .shx, and .dbf)
  2. Click counties on the map to select them (state data will be included automatically)
  3. Create named groups and add selected counties to them
  4. Export your groups to Excel when finished

Note: The .dbf file must contain state information for proper county identification.
